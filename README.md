# node-music-mvc

A pure node project using MVC architecture

### Clone 

- $ `git clone git@github.com:johnnynode/node-music-mvc.git --depth 1`

### Install

- $ `npm i` or `cnpm i` or `yarn install`
- $ `bower install`

### Run

- $ `npm start` or `npm run start`
